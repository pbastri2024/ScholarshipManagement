# Scholarship Management System

## 📖 Project Overview
The Scholarship Management System is a web-based application designed to simplify and automate the scholarship application and management process. It enables students to submit scholarship applications online while allowing administrators to review, verify, approve, and manage applications efficiently. The system reduces paperwork, improves transparency, and streamlines the overall scholarship management workflow.

## ✨ Features
- Student Registration and Secure Login
- Scholarship Application Submission
- Application Status Tracking
- Admin Dashboard
- Student Record Management
- Scholarship Verification and Approval
- Secure Authentication
- Responsive and User-Friendly Interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## 🎯 Project Objective
The primary objective of this project is to digitize the scholarship management process by providing an efficient, secure, and transparent platform for students and administrators. It minimizes manual work, enhances data accuracy, and improves the overall application processing experience.

## 🚀 How to Run the Project
1. Download or clone this repository.
2. Install XAMPP or WAMP Server.
3. Copy the project folder into the `htdocs` directory.
4. Import the MySQL database into phpMyAdmin.
5. Start the Apache and MySQL services.
6. Open your web browser and visit:
   ```
   http://localhost/ScholarshipManagement
   ```
7. Log in as a student or administrator to access the system.
