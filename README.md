# Scholarship Management System

## Project Overview
The Scholarship Management System is a web-based application developed to simplify the scholarship application and management process. It enables students to apply for scholarships online while allowing administrators to review, verify, and manage applications efficiently.

## Features
- Student Registration and Login
- Scholarship Application
- Application Status Tracking
- Admin Dashboard
- Student Record Management
- Secure Authentication
- Responsive User Interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Objective
The objective of this project is to digitize the scholarship process, reduce paperwork, improve transparency, and make scholarship management faster and more efficient.

## How to Run
1. Download or clone the repository.
2. Install XAMPP/WAMP.
3. Import the MySQL database.
4. Copy the project folder to the `htdocs` directory.
5. Start Apache and MySQL.
6. Open the project in your browser.
